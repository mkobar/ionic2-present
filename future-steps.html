Adding Android or iOS to an Ionic 2/3+ Project

I will build out my project for an Android phone, since that is what I own. In order to do that, you have to add android as a platform with cordova. You will need to add either android or ios, as this project will not work on a desktop environment.

For Android, you will need Android Studio with the Android-SDK, along with Java. 

For iOS, it is strongly suggested that you use a Mac with Xcode installed. Hoever these is a Windows option (see below).

There is a platform guide though that gives you all of the information you need for both iOS and Android:

  • Android Platform Guide
  • iOS Platform Guide

Once you have what you need installed, run: (or with ios)

$ cordova platform add android

Build for iOS on Windows

If you are really ready for a lot of work you can Build for iOS on Windows but it is a bit of work. A very good guide to do this can be found here: Building Ionic 2 Applications for iOS without a Mac

Running your Ionic App on a Mobile Phone

Connect your phone via USB cable (ensure debugging, the device's GPS, and all that jazz is on) and in the console, type:

$ ionic run android --device

It will take a good minute on the first run, and then the app will load up in the phone and if all goes according to plan, you will see a screen requesting GPS access.

Give it access to the GPS. Then you can click the GET POSITION button. 

Next you should see the alert with your current location.
